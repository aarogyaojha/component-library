# aarogya-ui

> The official CLI for [Aarogya UI](https://github.com/aarogyaojha/component-library) — a premium React component library built on Tailwind CSS v4 + Radix UI.

## Getting Started

```sh
npx aarogya-ui@latest init
```

## Commands

| Command | Description |
|---|---|
| `aarogya-ui init` | Initialise Aarogya UI in your project |
| `aarogya-ui add <name>` | Add a specific component |
| `aarogya-ui list` | List all available components |

## Available Components

| Category | Components |
|---|---|
| UI Primitives | `button`, `badge`, `table` |
| Layout | `card` |
| Feedback | `alert`, `dialog` |
| Forms | `input`, `textarea`, `checkbox`, `select`, `switch` |
| Navigation | `tabs` |

## Example

```sh
# 1. Initialize the library
npx aarogya-ui@latest init

# 2. Add individual components
npx aarogya-ui add button
npx aarogya-ui add dialog alert
```
